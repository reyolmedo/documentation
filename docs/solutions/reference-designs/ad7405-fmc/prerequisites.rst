Prerequisites
===============================================================================

The prerequisites depend on the AD7405 evaluation use case. For standard
evaluation using the supported SDP-H1 platform, the following items are
required.

Hardware prerequisites
-------------------------------------------------------------------------------

#. The AD7405 evaluation board:
   :adi:`EVAL-AD7405FMCZ`

#. The system demonstration platform used to control the evaluation board:
   :adi:`EVAL-SDP-CH1Z` (SDP-H1)

#. A suitable analog signal source capable of driving the AD7405 input
   within its specified input range.

#. USB cables:
  - USB cable provided with the SDP-H1 for connection to the host PC
  - 12 V power adapter provided with the SDP-H1

#. A Windows-based host PC with an available USB port.

Software prerequisites
-------------------------------------------------------------------------------

#. AD7405 evaluation software, available from the AD7405 product page or
   from the installer CD included with the evaluation kit.

#. EVAL-SDP-CH1Z (SDP-H1) USB drivers, installed as part of the evaluation
   software setup.
